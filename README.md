# Untitled Farming Game

Welcome to the Untitled Farming Game! This is a text-based adventure game where you can explore different worlds, fight enemies, and purchase items to enhance your gameplay experience.

##  Contents:

- Features
- Requirements
- Installation
- Game Controls

## Features

- Explore various worlds 
- Fight enemies and earn coins
- Purchase swords, armor, and power-ups from the shop
- Save and load your game progress


## Installation

1. **Clone the repository:**

   git clone https://github.com/AlessandroSanchi/Progetto.git


## How to start the program
Create a venv and activate it, and then install the requirements.
- python -m venv venv
- source venv/bin/activate 
- pip install -r requirements.txt

## Game Controls
- 1: Explore the selected world
- 2: Access the shop to buy items
- 3: Fight enemies
- 4: Quit the game
- 5: Save your game
- 6: Load your game
- Follow the on-screen prompts to navigate through the game.
